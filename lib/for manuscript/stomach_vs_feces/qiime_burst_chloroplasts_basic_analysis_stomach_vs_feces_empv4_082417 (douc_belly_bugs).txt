EMPV4 (R1 and R2)

Comparisons between datasets 
*The following comparisons are being made:
1. EMPV4_R1andR2_SZ_douc_stomach_contents and EMPV4_R1andR2_SZ_douc_feces (one_sample_per_individual_matching_subjects_with_fecal_samples)(sz_rsd_stomach_contents_vs_feces_comparison):

I. Making OTU biom table:

1. biom convert -i douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.txt -o douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.biom --table-type "otu table"   

2. biom summarize-table -i douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.biom -o otu_summary_douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.txt



II. Compute Core Microbiome: 

1. compute_core_microbiome.py -i douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.biom -o otu_table_core_jbcempv4_burst_douc_stomach_vs_feces_comparison_chloro



III. Alpha Diversity:

1. This following command is needed, as the alpha_params.txt file needs to be created manually: echo "alpha_diversity:metrics shannon,PD_whole_tree,chao1,observed_species" > alpha_params.txt

alpha_rarefaction.py -i douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.biom -m EMPV4_singapore_zoo_douc_stomach_contents_and_feces_matching_mapfile_082417.txt -o wf_arare_jbcempv4_burst_douc_stomach_vs_feces_comparison_chloro/ -p alpha_params.txt -t /home/knightsd/algh0022/sop/CHLORO.tre



VI. Beta Diversity:

beta_diversity_through_plots.py -i douc_stomach_vs_feces_chloro_QIIMEcompatibleOTUtable.biom -m EMPV4_singapore_zoo_douc_stomach_contents_and_feces_matching_mapfile_082417.txt -t /home/knightsd/algh0022/sop/CHLORO.tre -o wf_bdiv_jbcempv4_burst_douc_stomach_vs_feces_comparison_chloro/
