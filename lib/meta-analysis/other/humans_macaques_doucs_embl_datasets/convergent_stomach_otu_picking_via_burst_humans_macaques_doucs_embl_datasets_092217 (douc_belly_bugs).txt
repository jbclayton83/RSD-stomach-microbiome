Step 2: Align to 16S markerGMG version "PROK_170704", making a blast alignment file
burst12_old -r /home/knightsd/algh0022/sop/PROK_170704.edx -a /home/knightsd/algh0022/sop/PROK_170704.acx -b /home/knightsd/algh0022/sop/PROK_170704.tax -q convergent_evolution_stomach_meta-analysis_combined_humans_macaques_doucs_embl_datasets.fasta -o convergent_evolution_stomach_meta-analysis_aligns_humans_macaques_doucs_embl_datasets.b6 -fr -n -m CAPITALIST -bs -i 0.935 -f -sa

Step 4: Make OTU table compatible with QIIME tree (it substitutes "s" the underscore "/_" with a space "/ " globally "g" in the original table and outputs (">") the converted table)
sed 's/_/ /g' convergent_evolution_stomach_meta-analysis_otutable_humans_macaques_doucs_embl_datasets.txt > convergent_evolution_stomach_meta-analysis_QIIMEcompatibleOTUtable_humans_macaques_doucs_embl_datasets.txt
